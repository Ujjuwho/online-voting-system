# Online Voting System (C language, beginner style)

Hello!  
This is my simple C language project for learning purpose. It's a **console-based voting system**.  
It is made step by step, so you can see my early mistakes and how I fixed them.  
It has 4 versions showing how I learned programming in C.  

## What is this?
This is a menu-based voting system where you can vote for 3 candidates.  
At first, it's very simple and has bugs, but I try to improve after!

## How I Improved It
- First, it allowed everything (even mistakes!)
- Then I added input checking
- Then I made it so you can't vote twice in one run
- At the end, I made it a bit cleaner and user-friendly
